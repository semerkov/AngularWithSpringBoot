A barebone AngularJS project for the SDK
================================

Folder structure
----------------------------

- `index.html `: should fetch the `app.js` and `UserswebapiModule.js`
- `app.js `: should use the module `'UserswebapiModule'`
- `UserswebapiModule.js `: your SDK

Usage instructions
----------------------------

- Un-zip the downloaded file
- edit `app.js` and `index.html` to use the JavaScript SDK
 - Note: you can set a HTTP basic authentication by completing the run block in `app.js`
- serve it from any HTTP server, for example: https://www.npmjs.com/package/http-server with the
command line 
```
http-server /path/to/your/unzipped/sdk
```
- open the file index.html in your browser
- go on playing with your API :-)
